Daily
=====

Server side web frameworks collection.


## Implementations

... Send us a [pull request](https://github.com/bcho/daily/pulls) to add your cool daily blog!


## Requirements

Each implementation should provide:

- a usable application that:

  * is a blog (comment is not necessary) 
  * provides a editor for writing **online**
  * supports featured images for posts
  * uses our frontend & backend template & theme
  * has atom feed output
  * supports single author (multiple authors is welcomed)
  * other changes are welcomed

- a detailed deploy/develop guide


Feel free to use [vagrant](http://vagrantup.com) or [docker](https://www.docker.com).
